관련 공지 : https://saedu.naver.com/notice/view.naver?notiSeq=4261  <br>

1. 대상 기능 및 항목<br>
비즈채널 웹사이트 생성/수정시 businessInfo에 사이트 이름(siteName) 입력 요건이 추가 되었습니다.  <br> 

```json
{
    "name": "비즈채널",
    "channelTp": "SITE",
    "businessInfo": {
        "site": "https://naver.com",
        "siteName": "비즈채널"
    },
    "customerId": 0
}
```
